Ce dossier contient les fichiers OFF bruités.

Le suffixe de chaque fichier indique le type de bruit utilisé, ainsi que l'intensité du bruit.

Différent types de bruits : 
 - Gaussian
 - Aléatoire
 - Perlin
 - bruit blanc (moyenne nulle)
